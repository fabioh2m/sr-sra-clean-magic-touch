# 🧼 Sr. & Sra. Clean!

Aplicação web desenvolvida para apresentar serviços de **faxina profissional** e **personal chef**, com foco em experiência moderna, responsiva e conversão de clientes.

---

## 🔗 Acesse o projeto

👉 https://sr-sra-clean-magic-touch.vercel.app

---

## 📌 Sobre o projeto

Este projeto foi inicialmente estruturado com auxílio de ferramentas de geração automática, mas foi **totalmente ajustado, configurado e publicado manualmente**, incluindo:

* estruturação do ambiente com Vite + React
* correção de rotas e deploy
* configuração de build para produção
* publicação profissional utilizando Vercel

---

## ⚙️ Tecnologias utilizadas

* React
* Vite
* TypeScript
* Tailwind CSS
* React Router

---

## 🚀 Deploy

O projeto está hospedado na plataforma:

* Vercel (deploy automático via GitHub)

---

## 💻 Como rodar localmente

```bash
npm install
npm run dev
```

Praticar desenvolvimento frontend moderno, deploy e organização de projeto, simulando um cenário real de entrega para cliente.
